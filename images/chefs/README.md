## 사진 요구사항

- **형식**: JPG, PNG, WebP
- **권장 크기**: 500x500px 이상 (정사각형)
- **최대 파일 크기**: 500KB 이하 (웹 최적화)
- **품질**: 고화질, 얼굴이 잘 보이는 프로필 사진

## 사진 준비 방법

1. 각 셰프의 공식 프로필 사진 또는 방송 캡처 사용
2. 이미지 편집 툴로 정사각형으로 크롭
3. 파일명을 위의 정확한 이름으로 저장
4. 이 폴더(`images/chefs/`)에 업로드

## 저작권 주의사항

- 공식적으로 배포 가능한 사진만 사용하세요
- 넷플릭스 또는 제작사의 공식 이미지 사용 권한을 확인하세요
- 본인이 촬영한 사진이거나 저작권자의 허가를 받은 사진만 사용하세요

## 플레이스홀더 (임시 방법)

실제 사진이 준비되지 않은 경우, Unsplash에서 셰프/요리사 이미지를 임시로 사용할 수 있습니다:

```
https://images.unsplash.com/photo-1577219491135-ce391730fb2c?w=500&h=500&fit=crop (예시)
```

하지만 최종적으로는 실제 셰프들의 사진으로 교체해야 합니다.
