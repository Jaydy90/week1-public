Trust Route가 해결하는 문제

정보 과잉 속에서 선택 피로도를 줄이고, 검증된 신뢰와 이동 편의성을 한 화면에서 제공합니다.

신뢰 기준 선필터

미쉐린·유명인·흑백요리사 등 명확한 기준으로 후보군을 먼저 줄입니다.

현실적인 이동

거리·시간 기반으로 지금 갈 수 있는 곳만 남겨 결정 피로도를 없앱니다.

길찾기 완결

도보/대중교통/차량 중 최적 동선을 기본값으로 제안하고 바로 연결합니다.

내 주변 15분 후보

현재 위치 기준으로 신뢰 배지가 있는 식당만 먼저 보여주고, 이동 가능 시간으로 재정렬합니다.

정렬

가장 효율적인 이동수단을 기본값으로 제안하며, 지도 앱 딥링크로 즉시 전환됩니다.

신뢰 배지와 근거 카드

모든 카드에는 출처·확인일·근거 유형이 포함됩니다. 배지는 판매되지 않습니다.

근거 카드 구성

  • 출처 링크 + 확인일 + 근거 유형을 표준화합니다.
  • 배지 레벨(1~5)로 신뢰 강도를 표기합니다.
  • 방문/등재/운영 여부를 명확하게 구분합니다.
  • 정정 요청 시 즉시 갱신합니다.

미쉐린

가이드 등재 여부와 연도, 등급 정보를 명확히 표기합니다.

유명인 방문

방송·공식 SNS·인터뷰 기반 출처만 인증합니다.

흑백요리사

출연 셰프의 운영/협업 매장을 구분해 제공합니다.

흑백 요리사 맛집

출연 셰프가 운영하거나 협업한 매장을 서울 MVP 기준으로 먼저 공개합니다.

MVP 4화면 플로우

홈 → 리스트 → 상세 → 길찾기까지 한번에 완결되는 사용 흐름을 설계합니다.

01

홈

지도 중심 UI + 내 주변 10/15/30분 퀵 필터.

02

리스트

영업중/가격대/거리 기반 정렬과 신뢰 배지 필터.

03

상세

근거 카드 + 대표 메뉴 + 예상 이동 시간 + 저장/공유.

04

길찾기

도보/대중교통/차량 3탭 + 지도 앱 딥링크.

자주 묻는 질문

신뢰 기준과 업데이트 정책을 투명하게 공개합니다.

Trust Route는 어떤 기준으로 맛집을 선정하나요?

미쉐린 가이드, 유명인 방문, 흑백요리사 출연 등 공개된 근거를 확인하고 등록합니다.

출처가 불확실한 정보는 어떻게 처리하나요?

출처 확인 전에는 검증 중으로 표시하고, 확인일과 근거 링크를 확보한 후 업데이트합니다.

길찾기 동선은 어떻게 결정되나요?

도보/대중교통/차량 중 가장 효율적인 이동수단을 기본값으로 제안합니다.

배지는 유료로 부여되나요?

아니요. 배지 판매는 금지하며, 근거와 확인일이 확보된 경우에만 부여합니다.
